Jogo de Baralho comum de 1 jogador
- Objetivo: 
	Pegar todas as cartas do oponente
- Início: 
	Todas as cartas são distibuídas entre os jogadores
- Cada rodada:
	Cada jogador vira a carta do topo, o jogador dono da carta de maior valor fica com todas as outras e as coloca no fundo do próprio deck
	Em caso de empate, cada jogador coloca a própria carta no próprio deck
